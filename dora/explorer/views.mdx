---
title: Views
---

The Explorer section can toggle between two views, grid and list. The grid view is always shown as the default.
Most of the actions between both views are identical.

## Grid View
Grid view allows you to see your files and folders laid out in a grid format.

## List View
List view shows you your files and folders in list format.


### Mouse Actions
##### Folders
* Double-click each folder to navigate to it.
* Single-click each folder to focus on it.
* Right-click each folder to display folder options floater.

##### Files
* Double-click each file to open it with the default application.
* Single-click each file to focus on it.
* Right-click each folder to display file options floater.
