---
title: Mentions
---

Dora allows you to mention files and folders to add as context to the chat.
Mentioned files and folders will appear above your input in the chat box. To remove them from context, simply click on the `x` mark next to them.

## Mention From The Chat Window
<Frame>
    <img src="/assets/images/dora/chat/mentions1.png" />
</Frame>
You can mention files and folders right from your chat box by simply typing the letter `@`.
The letter `@` will be detected by Dora and display a floater list of files and folders in your current working directory.
Simply click on any of the items listed and they will be added to your chat session as context.


## Mention From The Explorer
<Frame>
    <img src="/assets/images/dora/chat/mentions2.png" />
</Frame>
Alternatively, you can mention files and folders simply by right-clicking on any of them add clicking "Mention in Chat".


## Mention From Model Response
<Frame>
    <img src="/assets/images/dora/chat/mentions3.png" />
</Frame>
Lastly, you can mention items from the model response itself if it returned relevant items to you.

---

### Limitations
All mentioned files and folders will be added to the context window as reference but not all of them will have their content parsed and digested by the model.
Size and file type limitations apply, however their path and name will be shared with the model for reference.

### Parsable File Types
* PDFs
* Code/Text
* Word Docs
